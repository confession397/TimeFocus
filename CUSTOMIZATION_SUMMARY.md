# TimeFocus 专注时钟 - 项目定制总结

## 一、项目概述

**项目名称**：TimeFocus 专注时钟（定制开源版）  
**原始项目**：WXminiprogram-Focus-clock（原作者：realyao）  
**定制版本**：V-1.0.0  
**开源许可**：MIT License

TimeFocus是一个基于微信小程序原生开发的时间管理工具，采用番茄工作法理念，帮助用户提高专注力和工作效率。

---

## 二、技术栈分析

### 2.1 核心技术
- **开发框架**：微信小程序原生开发
- **UI框架**：WEUI + VanUI组件库
- **图表库**：wxcharts.js
- **微信基础库版本**：>= 2.21.0
- **设备适配**：iPhone6及以上所有机型

### 2.2 技术架构
- **前端语言**：WXML（结构）、WXSS（样式）、JavaScript（逻辑）
- **数据存储**：wx.getStorageSync/wx.setStorageSync（本地存储）
- **API调用**：微信小程序原生API（计时、振动、背景音频等）

---

## 三、核心功能模块

### 3.1 专注计时器（主页面）
- **功能描述**：基于番茄工作法的专注计时功能
- **实现方式**：setInterval定时器 + 环形进度条动画
- **特性**：
  - 自定义任务名称
  - 工作/休息模式切换
  - 震动提醒
  - 计时进度可视化

### 3.2 待办清单管理
- **功能描述**：日常任务管理工具
- **实现方式**：本地存储 + 列表渲染
- **特性**：
  - 添加/删除任务
  - 完成状态切换
  - 批量操作（全选、清除已完成）
  - 任务统计

### 3.3 数据记录与统计
- **功能描述**：专注历史记录与数据分析
- **实现方式**：wxcharts.js环形图表 + 本地存储
- **特性**：
  - 工作时间/休息时间统计
  - 专注次数统计
  - 数据可视化展示
  - 历史记录查看

### 3.4 白噪声背景音效
- **功能描述**：专注时的背景音效播放
- **实现方式**：wx.getBackgroundAudioManager
- **音效类型**：
  - 林间（森林环境音）
  - 雨声（雨滴声）
  - 田野（自然环境音）
  - 静音（关闭背景音）

### 3.5 用户设置
- **功能描述**：个性化配置管理
- **实现方式**：本地存储持久化
- **配置项**：
  - 工作时间（1-60分钟）
  - 休息时间（1-30分钟）
  - 震动提醒开关
  - 背景音乐选择

## 四、设计模式分析

### 4.1 MVC架构模式
- **Model（数据层）**：Page.data中的数据状态 + Storage本地存储
- **View（视图层）**：WXML模板 + WXSS样式
- **Controller（控制层）**：Page中的事件处理函数和生命周期方法

### 4.2 本地存储模式
- **设计思想**：将用户数据持久化到本地Storage
- **优势**：
  - 无需服务器，降低开发成本
  - 数据安全性高（不依赖网络）
  - 访问速度快
- **应用场景**：
  - 用户设置（workTime、restTime、vibison）
  - 待办清单（todo_list）
  - 专注记录（logs）

### 4.3 定时器模式
- **实现方式**：setInterval + clearInterval
- **核心逻辑**：
  1. 记录开始时间
  2. 每秒更新剩余时间
  3. 计算进度百分比
  4. 更新环形进度条角度

### 4.4 组件化设计
- **UI组件**：WEUI提供的标准组件
- **自定义组件**：
  - 专注计时器（带动画）
  - 环形图表
  - 待办列表项
- **优势**：代码复用，维护性高

## 八、开源部署指南

### 8.1 环境准备
1. **安装微信开发者工具**：[下载地址](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. **注册微信小程序**：[注册地址](https://mp.weixin.qq.com/)
3. **获取AppID**：登录微信公众平台 -> 开发 -> 开发管理 -> 开发设置

### 8.2 项目部署步骤
1. **克隆项目**：
   ```bash
   git clone https://github.com/yourusername/TimeFocus.git
   ```

2. **打开项目**：
   - 使用微信开发者工具打开项目目录
   - 填入自己的AppID到`project.config.json`

3. **配置音乐资源**（可选）：
   - 将白噪声音频文件上传至CDN或服务器
   - 修改`pages/setting/setting.js`中的`musicUrl`数组

4. **编译运行**：
   - 点击工具栏"编译"按钮
   - 在模拟器或真机预览

5. **上传发布**：
   - 点击工具栏"上传"按钮
   - 填写版本号和项目备注
   - 在微信公众平台提交审核

### 8.3 注意事项
1. **音乐资源**：示例URL为占位符，需替换为实际可访问的音频地址
2. **AppID**：必须填入自己的AppID才能正常编译
3. **域名白名单**：如使用外部资源，需在微信公众平台配置服务器域名
4. **隐私协议**：发布前需配置用户隐私保护指引
