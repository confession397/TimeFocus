<p align=center>TimeFocus - 专注时间管理，提升工作效率的开源微信小程序</p>


## 🔎Features
* 专注计时（基于[番茄工作法](https://baike.baidu.com/item/%E7%95%AA%E8%8C%84%E5%B7%A5%E4%BD%9C%E6%B3%95/6353502)）
* 待办清单管理（ToDo List）
* 数据记录与图表分析
* 白噪声背景音效
* 自定义设置（工作时间、休息时间）
* 任务分享与记录导出

## 🚀快速开始
* **部署源码：**
```
1. 克隆仓库：git clone https://github.com/yourusername/TimeFocus.git
2. 使用微信开发者工具打开项目
3. 在 project.config.json 中填入你的 appid
4. 编译运行
```
* **本项目是基于微信小程序原生开发，涉及基础的 JavaScript、WXML、CSS 等前端知识，适合新手学习入门、毕业设计课题。**
  * 开发前请参考[微信小程序官方文档](https://developers.weixin.qq.com/miniprogram/dev/framework/)。
  * 项目使用[WEUI](https://github.com/Tencent/weui-wxss)组件库，提供微信原生视觉体验。
  * 微信基础库版本要求：>=`2.21.0`，适配 iPhone6 及以上机型。
  * 已移除所有第三方服务密钥和个人数据，确保安全开源。
* **本开源项目基于原项目进行个性化定制和安全性改造，所有敏感信息已移除或替换。**
* **欢迎提交 Issue 和 Pull Request！**
